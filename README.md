# React Product Cards

A React exercise that renders a product catalogue from data using reusable, composable components.

## Features

- Product data mapped into reusable components
- Dedicated `Name`, `Image`, `Price` and `Description` components
- Demonstrates JSX composition and props passing

## Getting started

```bash
git clone https://github.com/Ayinde225/react-product-cards.git
cd react-product-cards
npm install
npm start
```

Then open http://localhost:3000 in your browser.

## Tech stack

React, JSX, JavaScript, Create React App
